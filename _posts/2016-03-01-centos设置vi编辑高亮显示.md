CentOS里只默认安装了vi，没有vim,所以syntax高亮不起作用。#下载vim[root@zkyml ~]#...

> CentOS里只默认安装了vi，没有vim,所以syntax高亮不起作用。

**下载vim**

[root@zkyml ~]# <span style="color: #0000ff;">yum -y install vim</span>

**编辑bash配置设置vim的别名为vi**

[root@zkyml ~]# <span style="color: #0000ff;">vi /etc/bashrc</span>
```sh
alias vi='vim'
```
**编辑vim的配置,显示行号**

[root@zkyml ~]# <span style="color: #0000ff;">vi /etc/vimrc</span>
```sh
set nu

syntax on #如果有就不需要了
```
**重新加载生效**

[root@zkyml ~]# <span style="color: #0000ff;">source /etc/bashrc</span></pre>
