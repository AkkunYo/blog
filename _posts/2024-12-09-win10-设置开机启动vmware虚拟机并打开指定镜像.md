#### （1）将VMware的快捷方式放到系统启动文件夹下
```
#启动路径
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
```
#### （2）修改快捷方式属性中的目标栏目

- 右键VMware Workstation快捷方式，看属性

- 在目标中直接后续添加
```
#规则
-x "vmx_path"
```
目标中完整内容为
```
#我的示例
"E:VMwareVMware Workstationvmware.exe" -x "E:VMwareVirtual Machinescs7.6CentOS 7 64 位.vmx"
```
